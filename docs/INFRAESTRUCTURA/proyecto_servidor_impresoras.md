# Proyecto - Servidor de impresoras

## Estado

Planificacion inicial.

## Objetivo

Centralizar la administracion de impresoras para mejorar control, disponibilidad y soporte.

## Alcance inicial

- Inventario de impresoras actuales.
- Definicion de equipos cliente y perfiles de uso.
- Configuracion de servidor de impresion.
- Prueba piloto por area.

## Entregables esperados

- Procedimiento de alta/baja de impresoras.
- Guia de instalacion para equipos cliente.
- Politica de uso y mantenimiento.
- Plan de contingencia basico.

## Pendientes

- Definir plataforma del servidor.
- Definir esquema de autenticacion.
- Definir responsables operativos.
