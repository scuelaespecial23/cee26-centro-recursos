# Guia breve para docentes

## Para que sirve este repositorio

Este espacio concentra guias, modelos y recursos de trabajo para facilitar tareas institucionales.

## Que vas a encontrar

- Plantillas y documentos comunes.
- Guias paso a paso.
- Scripts de automatizacion (uso coordinado con TIC).

## Como usarlo en el dia a dia

1. Revisar el indice general en `docs/INDICE.md`.
2. Ir a la seccion correspondiente (guias, normativa, recursos).
3. Descargar o reutilizar el material necesario.
4. Si falta algo, abrir una solicitud en Issues.

## Recomendacion

Evitar editar directamente recursos originales. Crear copias de trabajo para cada caso.
