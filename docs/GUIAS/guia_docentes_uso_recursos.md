# Guia breve para docentes

Ultima actualizacion: 2026-03-30

## Para que sirve este repositorio

Este espacio concentra recursos institucionales para que no dependan de una sola computadora, de un mensaje viejo o de la memoria de una persona.

## Que vas a encontrar

- guias de uso,
- borradores de normativa,
- scripts para tareas repetitivas,
- recursos compartidos y modelos de trabajo.

## Como conviene usarlo

1. Entrar por `docs/INDICE.md`.
2. Buscar el tema por seccion: guias, normativa, scripts o recursos.
3. Leer primero la guia antes de ejecutar un script o reutilizar un modelo.
4. Si un documento es plantilla, trabajar sobre una copia y no sobre el original.

## Recomendaciones practicas

- Mantener nombres claros en archivos y carpetas.
- Evitar duplicar versiones del mismo documento con nombres confusos.
- Si un procedimiento se repite mucho, pedir que quede documentado.
- Si algo falta o esta desactualizado, registrarlo para corregirlo.

## Donde mirar segun necesidad

- Si necesitas una ayuda paso a paso: `docs/GUIAS/`.
- Si necesitas una regla o criterio comun: `docs/NORMATIVA/`.
- Si necesitas automatizar una tarea: `scripts/`.
- Si necesitas modelos o materiales base: `recursos_compartidos/`.
