# Guia paso a paso para Epson L220 en Windows

Ultima actualizacion: 2026-04-29

## Objetivo

Dejar documentado el procedimiento para instalar, configurar y probar una impresora Epson L220 en Windows usando los recursos oficiales de Epson.

## Enlace oficial de soporte

Para buscar drivers y utilidades oficiales en Windows, usar:

- Soporte Epson Argentina: `https://epson.com.ar/Soporte/Impresoras/sh/s1`

## Antes de empezar

- Tener la impresora apagada o desconectada si el instalador lo recomienda.
- Tener a mano el cable USB.
- Confirmar que el equipo tenga acceso a Internet.
- Verificar que la impresora tenga papel y tinta.

## Paso 1. Buscar el modelo correcto

1. Abrir el sitio de soporte de Epson Argentina.
2. En el buscador de impresoras, escribir `L220`.
3. Entrar en la pagina del modelo `Epson L220 Series`.
4. Elegir la version de Windows correspondiente.

Ejemplos posibles:

- Windows 10
- Windows 11
- 64 bits o 32 bits, segun el equipo

## Paso 2. Descargar el software necesario

Se recomienda descargar, como minimo:

1. El driver principal de la impresora.
2. La utilidad de Epson si esta disponible para el sistema.

Si Epson ofrece un paquete combinado de instalacion, usar ese paquete como opcion principal.

## Paso 3. Ejecutar el instalador

1. Abrir el archivo descargado.
2. Aceptar permisos de administrador si Windows los solicita.
3. Seguir el asistente de instalacion.
4. Aceptar el acuerdo de licencia.
5. Cuando el instalador lo pida, conectar y encender la impresora por USB.

## Paso 4. Confirmar que Windows detecte la impresora

Al finalizar la instalacion, revisar en:

1. `Configuracion`
2. `Bluetooth y dispositivos`
3. `Impresoras y escaneres`

La `Epson L220 Series` debe aparecer en la lista.

## Paso 5. Imprimir pagina de prueba

1. Entrar en `Configuracion > Bluetooth y dispositivos > Impresoras y escaneres`.
2. Seleccionar `Epson L220 Series`.
3. Entrar en `Preferencias de impresion` o `Propiedades de impresora`.
4. Ejecutar `Imprimir pagina de prueba`.

## Paso 6. Si la hoja sale en blanco

Si la impresora responde pero imprime hojas en blanco:

1. Confirmar que el driver descargado corresponde al modelo `L220`.
2. Revisar si Windows asigno una impresora generica en lugar del driver Epson.
3. Si hay una configuracion previa defectuosa, quitar la impresora y volver a instalarla.
4. Verificar niveles de tinta.
5. Ejecutar mantenimiento de cabezales.

## Paso 7. Limpieza de cabezales en Windows

La opcion puede aparecer como `Mantenimiento`, `Nozzle Check`, `Head Cleaning` o similar.

Ruta habitual:

1. `Configuracion`
2. `Impresoras y escaneres`
3. `Epson L220 Series`
4. `Preferencias de impresion` o `Propiedades`
5. `Mantenimiento`

Luego ejecutar en este orden:

1. `Nozzle Check`
2. `Head Cleaning`
3. Nueva pagina de prueba

## Recomendaciones de mantenimiento

- No repetir limpiezas muchas veces seguidas.
- Si la impresion sigue entrecortada, esperar 20 a 30 minutos antes de repetir.
- Si la impresora estuvo mucho tiempo sin uso, puede requerir limpieza adicional.

## Cuando conviene reinstalar desde cero

Conviene desinstalar y volver a instalar si ocurre alguno de estos casos:

- la impresora aparece con nombre generico,
- la pagina sale en blanco aun con tinta,
- la impresion queda en cola pero no sale,
- el equipo reconoce el USB pero no figura la Epson L220 como impresora.

## Resumen corto del procedimiento recomendado

1. Entrar al soporte oficial de Epson Argentina.
2. Buscar `L220`.
3. Descargar el driver correcto para la version de Windows.
4. Ejecutar el instalador como administrador.
5. Conectar la impresora cuando el asistente lo solicite.
6. Imprimir una pagina de prueba.
7. Ejecutar limpieza de cabezales si la impresion sale incompleta.
