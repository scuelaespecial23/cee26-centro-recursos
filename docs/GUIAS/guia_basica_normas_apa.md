# Guia basica de normas APA para documentos institucionales

Ultima actualizacion: 2026-03-30

## Objetivo

Dejar un criterio minimo y comun para que los documentos institucionales tengan mejor presentacion, orden y consistencia.

## Formato recomendado

- Hoja A4.
- Margenes de 2,54 cm en los cuatro lados.
- Fuente legible: Arial 11, Calibri 11 o Times New Roman 12.
- Interlineado doble si el documento es formal o academico.
- Alineacion a la izquierda.
- Sangria de primera linea en parrafos largos cuando corresponda.

## Portada basica sugerida

Incluir, segun el tipo de documento:

- nombre de la institucion,
- titulo del documento,
- area, docente o responsable,
- fecha,
- curso, grupo o destinatario si aplica.

## Titulos y subtitulos

- Usar titulos claros y consistentes.
- Evitar mezclar demasiados tamanos y estilos.
- Mantener el mismo criterio en todo el documento.

## Citas breves

Si se cita textual o se toma una idea de otra fuente, indicar autor y ano.

Ejemplo:

`Segun Perez (2022), la inclusion requiere planificacion institucional.`

O bien:

`La inclusion requiere planificacion institucional (Perez, 2022).`

## Lista de referencias

Al final del documento, incluir una seccion `Referencias` si se usaron fuentes externas.

Ejemplo de libro:

`Apellido, A. A. (2022). Titulo del libro. Editorial.`

Ejemplo de pagina web:

`Organizacion. (2024). Titulo del recurso. https://sitio-ejemplo.org`

## Recomendaciones para la escuela

- No sobrecargar los documentos con colores o tipografias decorativas.
- Usar negrita solo para destacar titulos o conceptos puntuales.
- Revisar ortografia antes de compartir o imprimir.
- Guardar una version editable y otra en PDF cuando el documento quede cerrado.

## Alcance

Esta guia no reemplaza el manual APA completo. Sirve como base practica para ordenar documentos institucionales hasta definir una version mas formal.
