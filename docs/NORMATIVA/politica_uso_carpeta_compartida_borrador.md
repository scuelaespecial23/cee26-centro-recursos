# Politica de uso de carpeta compartida - borrador

Ultima actualizacion: 2026-05-08

## Objetivo

Definir criterios basicos para el uso responsable del servidor de archivos institucional y su carpeta compartida.

## Alcance

Aplica a personal docente, equipo directivo, secretaria, equipo tecnico y cualquier otra persona autorizada a utilizar el recurso compartido institucional.

## Principios generales

- El servidor de archivos es un recurso institucional, no personal.
- El espacio compartido debe priorizar materiales utiles para la escuela.
- El uso responsable ayuda a conservar orden, espacio y accesibilidad.
- Toda persona usuaria debe respetar el trabajo y los archivos de las demas.

## Usos permitidos

Se considera adecuado subir:

- documentos institucionales,
- materiales pedagogicos relevantes,
- planillas de trabajo,
- evidencias seleccionadas de actividades escolares,
- archivos que sea importante conservar para consulta posterior.

## Usos no recomendados o no permitidos

No corresponde usar el servidor para:

- almacenar archivos personales ajenos a la institucion,
- subir grandes cantidades de fotos o videos sin revision previa,
- conservar multiples copias identicas del mismo archivo,
- guardar material borroso, defectuoso o sin utilidad real,
- usar el espacio como respaldo completo del celular,
- borrar o modificar archivos de otras personas sin autorizacion.

## Criterios para fotos y videos

Antes de subir imagenes o videos, se espera que cada docente realice una seleccion minima.

Ejemplos de buena practica:

- si hay fotos borrosas, no subirlas,
- si hay varias fotos casi iguales, elegir la mejor,
- si hay muchas tomas repetidas, conservar solo las necesarias,
- si el video no aporta valor institucional, no subirlo,
- si el material ya fue publicado y no necesita resguardo adicional, evaluar si realmente corresponde archivarlo.

## Uso del espacio docente compartido

La carpeta `01-docentes` funciona como espacio comun entre docentes habilitados.

Por ese motivo:

- cada docente deberia crear y usar su propia carpeta personal,
- no deberian tocarse archivos ajenos salvo acuerdo explicito,
- se espera una convivencia basada en responsabilidad adulta y cuidado comun.

Ejemplo de organizacion sugerida:

- `01-docentes/maria_jose`
- `01-docentes/juan_perez`

## Espacio publico

La carpeta `00-publico` puede utilizarse para materiales que tenga sentido compartir mas ampliamente dentro de la institucion.

No deberia usarse para depositar archivos sin clasificar o material de trabajo individual que corresponde a carpetas personales.

## Credenciales y acceso

- Las credenciales de acceso se solicitan al referente TIC.
- No corresponde difundir usuarios o contrasenas por canales no autorizados.
- El acceso remoto al servidor se habilitara solo en casos puntuales y con autorizacion expresa.

## Acceso remoto excepcional

Cuando exista una necesidad concreta de trabajo remoto, la persona interesada debera solicitarlo al referente TIC.

El procedimiento previsto incluye:

1. pedido de autorizacion,
2. instalacion de `Tailscale`,
3. validacion del equipo,
4. habilitacion del acceso si corresponde.

## Usuarios individuales y directorios privados

La institucion podra evaluar, en casos puntuales, la creacion de usuarios individuales y directorios no compartidos con otros usuarios.

Esto podra considerarse cuando exista:

- necesidad laboral concreta,
- uso sostenido y ordenado del recurso,
- compromiso con las pautas institucionales de cuidado del espacio.
